⚙️ 03 — Methodology

How It Works

A complete deep learning pipeline — from raw fingerprint to blood group classification in under 2 seconds.

Image Ingestion
Fingerprint uploaded via Flask REST API, stored in memory for immediate processing.
Preprocessing
Resize to 64×64, RGB normalization to [0,1], histogram equalization for consistency.
CNN Inference
4-block deep CNN extracts spatial ridge features. Softmax outputs probability across 8 classes.
Result Delivery
Blood group + confidence score returned as JSON and rendered instantly in the UI.
