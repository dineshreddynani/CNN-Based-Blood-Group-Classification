Architecture

How It Works

A full deep learning pipeline from fingerprint image to blood group prediction.

01
Image Upload
User uploads a fingerprint image via Flask REST API endpoint.
02
Preprocessing
Resize to 64×64, RGB normalization, histogram equalization for consistency.
03
CNN Inference
4-block deep CNN extracts ridge features → Softmax outputs probability across 8 classes.
04
Result Returned
Blood group + confidence score returned as JSON and displayed instantly.
