AI Blood Group Detection

Fingerprint-based deep learning prediction system

Upload Fingerprint

How It Works

This system uses a Convolutional Neural Network (CNN) trained on fingerprint images to classify blood groups.

When you upload a fingerprint image, it is resized to 64×64 pixels and normalized before being passed to the trained deep learning model.

The CNN extracts unique ridge pattern features from the fingerprint and compares them against patterns learned during training.

The model outputs probabilities for all eight blood groups (A+, A-, B+, B-, AB+, AB-, O+, O-).

The system selects the class with the highest probability and displays both the predicted blood group and confidence score.

The model is deployed on a cloud-based server using Flask and served via Gunicorn for scalable inference.
